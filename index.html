The Power B2X

Power Bank USB Portable Charging

The Power B2X is an affordable Power Bank with 2 USB ports and 1 USB charging port. It features a special type C port for Type C to Type C or Type C to any port connections. With 8000mAh capacity, it offers long battery life and a single charge multiple use framework to make all things work seamlessly. Just plug it and Charge away! The compact design is also pocket friendly, fitting in most pocket sizes for true portability.
